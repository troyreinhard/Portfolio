# Troy Reinhard — Portfolio

Static export of the portfolio site, ready for GitHub Pages.

## Deploy to GitHub Pages
1. Create a new GitHub repo and push this folder's contents to it (index.html at repo root).
2. In the repo, go to Settings → Pages.
3. Under "Build and deployment", set Source to "Deploy from a branch", branch `main`, folder `/ (root)`.
4. Save — your site will be live at `https://<username>.github.io/<repo>/` within a minute or two.

No build step needed — `index.html` is fully self-contained.
